# LangChain Vectors

Conexión a GPT utilizando LangChain con vectores como contexto (RAG).

## Tecnologías

- TypeScript
- LangChain
- OpenAI API
- Vector Store

## Instalación

```bash
npm install
```

Crear un archivo `.env` con tu API key:

```
OPENAI_API_KEY=tu_api_key
```

## Ejecución

```bash
npm start
```
